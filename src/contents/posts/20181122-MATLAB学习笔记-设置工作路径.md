# MATLAB学习笔记 —— 设置工作路径
工作路径作为不常用的基本操作，很容易被遗忘，所以记录一下。
# 目录

[TOC]



## 1 查看搜索路径
path指令
调用格式：
```
path
```

## 2 查看当前路径
cd指令
调用格式：

```
cd
```

## 3 设置工作路径
userpath()指令
调用格式：

```
userpath('需要设置的工作路径')
```
输入cd确认已经修改

```
cd
```
确认无误后保存默认搜索路径

## 4 保存工作路径

```
savepath
```
